# Design System

## Personality

Professional, calm, fresh, and academic. The site should look like it belongs to a serious early-career scholar: clear enough for search committees, inviting enough for collaborators, and polished enough to feel intentional.

## Visual Principles

- Use whitespace as the main luxury.
- Keep typography sharp and readable.
- Prefer simple section bands over stacked decorative cards.
- Make research outputs easy to skim.
- Avoid heavy gradients, oversized marketing language, and overly playful effects.

## Color Palette

- Background: `#f8f7f3`
- Surface: `#ffffff`
- Text: `#242420`
- Muted text: `#6b6b61`
- Primary accent: `#1f6f68`
- Secondary accent: `#b55d4c`
- Border: `#dedbd1`

## Typography

Recommended web font pairing:

- Headings: `Source Serif 4`, `Georgia`, serif
- Body: `Inter`, `Arial`, sans-serif

If avoiding external font loading, use:

```css
font-family: Inter, Arial, sans-serif;
```

and:

```css
font-family: Georgia, "Times New Roman", serif;
```

## Components

- Header: slim, sticky optional, high contrast text.
- Hero: split between identity text and a real headshot.
- Link buttons: simple border or filled accent buttons.
- Publication item: title, venue/status, authors, abstract-level summary, links.
- Footer: compact, with contact and profile links.

## Imagery

Use one excellent headshot. It should be:

- Well-lit
- Recent
- Cropped around upper body or shoulders
- Neutral or lightly academic background
- Saved as `assets/headshots/yibin-fan-headshot.jpg`

Optional visual assets:

- Research presentation photo
- Conference talk photo
- Data visualization image from research
